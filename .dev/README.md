# Entwicklungsdateien

Dieses Verzeichnis enthält interne Entwicklungsdokumente und Logs, die nicht Teil der Produktionsdokumentation sind.

## Inhalt

- **Plan.md** - Ursprünglicher Architekturplan
- **IMPLEMENTATION_LOG.md** - Protokoll der Implementierungsschritte
- **Entwicklungsland.md** - Entwicklungsnotizen und aktuelle Arbeiten
- **logs/** - Anwendungs- und Entwicklungslogs
- **sqlite_mcp_server.db** - Lokale Entwicklungsdatenbank

Diese Dateien sollten nicht in der Produktion verwendet werden.
