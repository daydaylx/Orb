# Dokumentation

Struktur:

- `spec/` — Formale Spezifikationen (z.B. `orb-config-spec.md`).
- `guides/` — Anleitungen zur Integration und Nutzung.
- `changelog/` — Kurzübersicht der Änderungen (Details in `.dev/IMPLEMENTATION_LOG.md`).

Einstieg:
- Für das Konfigurationsformat: `spec/orb-config-spec.md`.
- Für die Einbindung in Projekte: `guides/integration-guide.md`.
